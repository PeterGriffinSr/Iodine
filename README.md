# Iodine
Iodine is a lightweight and expressive scripting language built using OCaml.


# Features
- **Minimal Syntax**: Simple and easy-to-read syntax.

- **Built with OCaml**: Leveraging the OCaml ecosystem for a robust foundation.

- **Lightweight Dependencies**: Only requires dune and ppx_deriving.

- **Extensible**: Easily modifiable to suit specific needs.

# Prerequisites

To build and run Iodine, ensure you have the following installed on your system:

- **OCaml** (version >= 5.2.1)

- **Dune** (build system for OCaml)

- **ppx_deriving** (for deriving code generation)
